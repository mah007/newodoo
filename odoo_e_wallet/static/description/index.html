Odoo e-Wallet

Add website wallet to the Odoo website to transfer money directly to their Odoo website account. Make use of Odoo website e-wallet to provide store credit and bonuses to the customers or Odoo users.

Odoo admin can add the money to any customer's Odoo website e-wallet which can then be used by the customer for paying for his/her orders making the transaction process secure, fast and hassle free.

If set to true by the Odoo admin, the customers can view the transaction details of their Odoo e-wallet in their Odoo website accounts.

Features

  • The module adds a website wallet in Odoo.
  • The Odoo admin can add money to the Odoo e-wallet of Odoo users and customers.
  • The customers can use the wallet money on Odoo website to pay for their orders.
  • Reasons for adding balance to the wallet can be added to the form.
  • The admin can allow the customers to view the transactions of their Odoo e-wallet in their Odoo website account.
  • The wallet balance is automatically converted to the current price list of the Odoo website. For example; if the price list is changed from dollars to euro, the wallet balance automatically reflects the euro equivalent of the present balance.

Display/Hide Odoo e-wallet

Manage multiple wallets for multiple companies(One Per Company).

The module provide options to show/hide Odoo e-wallet and e-wallet transactions for customers and users.

Create a new e-wallet transaction

The Odoo admin can create a new transaction to add money to the Odoo e-wallet of the customer.

E-wallet transaction history

The details for each e-wallet transaction- credit and debit can be viewed in the Odoo backend.

Assigning transaction tags

The transaction tags can be created and assigned to each credit e-wallet transaction to show the reason for adding the money to the wallet such as bonus, exchange, refund, etc,

Wallet balance

The Odoo admin can view the wallet balance of the customers in the Odoo backend.

Paying with e-wallet on Odoo website

The Odoo admin can view the wallet balance of the customers in the Odoo backend.The customers can use their e-wallet balance to pay for the order. The remaining amount (if any) can be paid by any other method.

E-wallet transaction ID in order

Odoo e-wallet transactions details

The customers can view the details of their e-wallet transactions in their Odoo website accounts.
